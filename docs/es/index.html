Funciones

Todo para el crecimiento de tu hijo

Simple e intuitivo — diseñado para que los padres registren y analicen el desarrollo de estatura fácilmente

📈

Visualización de curvas de crecimiento

Renderizado vectorial en tiempo real de curvas de percentiles de 0 a 18 años. Compatible con zoom y muestra la estatura medida de tu hijo sobre la curva.

🎯

Análisis de percentiles

Calcula con precisión el percentil de tu hijo entre sus pares usando el algoritmo LMS. Detecta desviaciones del desarrollo de forma temprana.

🔮

Predicción de estatura adulta

Basándose en las tendencias de crecimiento actuales, predice científicamente el rango de estatura adulta esperada a los 18 años.

👨‍👩‍👧‍👦

Múltiples perfiles de hijos

Gestiona registros de crecimiento independientes para múltiples hijos simultáneamente, cada uno con sus propios datos y curva de crecimiento.

🌍

Múltiples fuentes de datos

Cambia entre OMS, Japón MHLW y estándares nacionales de China para comparar diferencias de percentiles entre referencias.

🔒

Almacenamiento 100% local

Todos los datos se almacenan únicamente en tu dispositivo. Sin cuenta, sin datos enviados a ningún servidor. La privacidad de tu hijo está completamente protegida.
