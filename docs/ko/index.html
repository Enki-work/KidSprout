주요 기능

아이의 성장을 위한 모든 것

간결하고 직관적인 인터페이스로 부모가 쉽게 키 발육을 기록하고 분석할 수 있습니다

📈

성장 곡선 시각화

권위 있는 통계 데이터를 기반으로 0~18세 백분위 성장 곡선을 실시간 벡터로 렌더링합니다. 확대/축소 지원, 실측 키를 곡선 위에 겹쳐 표시합니다.

🎯

백분위 분석

LMS 알고리즘으로 또래 중 아이의 키 백분위를 정확히 계산합니다. 상대적 위치를 직관적으로 파악하고 발육 이상을 조기 발견할 수 있습니다.

🔮

18세 키 예측

현재 성장 추세를 기반으로 18세 성인 예측 키 범위를 과학적으로 산출합니다.

👨‍👩‍👧‍👦

여러 자녀 프로필 관리

여러 자녀의 독립적인 성장 기록을 동시에 관리합니다. 각 자녀마다 별도의 데이터와 성장 곡선을 보유합니다.

🌍

다중 데이터 소스

WHO, 일본 후생노동성, 중국 국가 기준 간을 언제든지 전환하여 참고 기준별 백분위 차이를 비교할 수 있습니다.

🔒

완전 로컬 저장

모든 데이터는 기기에만 저장됩니다. 계정 불필요, 서버 업로드 없음. 아이의 개인정보가 완전히 보호됩니다.
