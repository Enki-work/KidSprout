개인정보 처리방침

최종 업데이트: 2026년 3월

💚 핵심 약속

KidSprout은 어떠한 개인 데이터도 수집, 저장 또는 전송하지 않습니다. 모든 데이터는 기기 로컬에만 저장됩니다.

아이의 성장 정보는 매우 민감한 가족 데이터임을 깊이 인식합니다. KidSprout은 처음부터 "데이터 수집 제로" 원칙으로 설계되었습니다.

📱 데이터 저장

  • 자녀 프로필(이름, 성별, 생년월일)은 기기에만 저장됩니다
  • 모든 키 측정 기록은 기기에만 저장됩니다
  • 언어 설정은 기기에만 저장됩니다
  • 앱 삭제 시 모든 로컬 데이터가 영구 삭제됩니다

🚫 수집하지 않는 정보

  • 개인 식별 정보(이름, 이메일, 전화번호 등) 미수집
  • 위치 정보 미수집
  • 기기 식별자 또는 광고 ID 미수집
  • 사용 행동 또는 통계 데이터 미수집
  • 제3자 분석 또는 광고 SDK 미사용

🌐 네트워크 요청

KidSprout은 어떠한 네트워크 요청도 하지 않습니다. 모든 성장 곡선 데이터(WHO / 일본 후생노동성 / 중국 기준)는 앱 내에 오프라인으로 내장되어 있어 인터넷 없이도 완전히 사용 가능합니다.

⭐ 앱 내 평가 기능

"앱 평가하기" 기능은 Apple App Store / Google Play 공식 네이티브 다이얼로그로 구현됩니다. 평가 처리는 플랫폼이 담당하며, KidSprout은 평가 데이터에 전혀 접근하지 않습니다.

👶 아동 개인정보 보호 (COPPA)

KidSprout은 성인(부모/보호자)이 사용하도록 설계되었습니다. 아동으로부터 직접 데이터를 수집하지 않으며, COPPA(아동 온라인 개인정보 보호법)를 준수합니다.
